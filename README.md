## AWS

### How to install AWS CLI in Mac

Step1: curl "https://awscli.amazonaws.com/AWSCLIV2.pkg" -o "AWSCLIV2.pkg"

Step2: sudo installer -pkg AWSCLIV2.pkg -target /

Step3: which aws

Step4: aws configure
