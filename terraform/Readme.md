
### Steps to install terraform in Mac OS

```
brew tap hashicorp/tap
```

```
brew install hashicorp/tap/terraform
```

### Steps to install tfenv in Mac OS

Reference: - https://opensource.com/article/20/11/tfenv

```
brew install tfenv 
```
